Transaction performance

Download data about the performance of transactions used to deliver Government services.

Citizens make use of Government services by interacting with one or more transactions through online or offline channels. A transaction is the whole process that a person needs to go through to create, update, or delete the information that Government holds about an individual, legal entity or physical item.

About the data

The data is provided as a CSV file. A line in the file represents one transaction and is made up of data items, each of which is either a characteristic, metric or some other supporting information about the transaction.

Characteristics are useful for sorting, filtering and aggregating data in the file, and for comparing metrics across transactions on a like-for-like basis.
